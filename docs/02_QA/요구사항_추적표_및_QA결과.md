## 데이터 QA

- 원본 600건: 고용24 100, 잡코리아 500
- 제외: 삭제/비활성 67, 비경기 고용24 73
- 최종 460건: 고용24 27, 잡코리아 433
- 검증: ID·공고·URL 중복 0, 필수 결측 0, 좌표/날짜/코드 형식 오류 0
- 상세 결과: `data/processed/data-quality-report.json`
- 좌표 연결: `MAP_COOR_Y=위도`, `MAP_COOR_X=경도`; 별도 기업 테이블이므로 `BUSINO ↔ BIZ_NO` 일치 건만 사용

## 자동 테스트 범위

- 전체/상세/404 조회, 안정 페이징
- bbox, 반경 및 거리순 조회
- 출처·키워드·경력·학력·고용형태·직종·희망임금 필터
- 관리자 인증/통계와 공개 API 접근
- Cloudflare IP 위치 헤더 및 헤더 미제공 폴백
- Spring 애플리케이션 컨텍스트

## 배포 전 체크

- 데이터 빌드와 품질 검증 통과
- Gradle 전체 테스트 통과
- API 전체 건수와 고유 ID 일치
- 운영 `/api/jobs`, `/api/location`, 정적 화면 응답 확인
- Docker Compose 컨테이너 상태 확인
