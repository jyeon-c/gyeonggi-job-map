## 남은 운영 설정 1건

Cloudflare 방문자 위치 헤더가 아직 원 서버로 전달되지 않아 PC IP 위치는 브라우저 위치로 폴백한다.

1. Cloudflare Dashboard에서 `jobmapkorea.com` 선택
2. Rules → Transform Rules → Managed Transforms
3. `Add visitor location headers` 활성화
4. `https://jobmapkorea.com/api/location`에서 `available=true`, `source=cloudflare-ip` 확인

코드·테스트·폴백은 배포 완료됐으며 이 설정은 Cloudflare 계정 권한이 필요한 운영 작업이다.
