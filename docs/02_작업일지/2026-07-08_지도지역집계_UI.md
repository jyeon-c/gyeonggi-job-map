# 지도 지역 집계 UI 개선 (2026-07-08)

- 지도에 개별 채용공고 핀 대신 현재 지도 범위·검색·필터 결과를 첫 근무지역 기준으로 묶어 `공고 수 + 지역명` 배지로 표시했다.
- 여러 공고가 있는 지역 배지는 클릭 시 해당 위치로 확대하고, 단일 공고 배지는 기존처럼 공고를 선택하도록 했다. 카드 선택 시 소속 지역 배지도 선택 상태로 강조된다.
- PC에서 왼쪽 공고 카드에 마우스를 올리면 지도에 해당 좌표를 반투명 원으로 표시한다. 정밀 좌표는 700m, 지역 근사 좌표는 2.2km 반경으로 정확도 차이를 반영했다.
- 지도 레벨 3 이하로 아주 가깝게 확대했을 때만 지역 집계 배지를 공공/민간 개별 공고 핀으로 전환하고, 지도 우측에 `+ / −` 확대·축소 버튼을 추가했다.
- 개별 핀 클릭 시 왼쪽 목록을 해당 공고 1건으로 좁히고, 휠이나 버튼으로 줌 레벨이 바뀌면 현재 지도 범위의 전체 목록으로 자동 복원하도록 상태 흐름을 보완했다.
- 개별 핀이 표시되는 지도 레벨 3 이하에서는 카드 hover 반경 원을 생략해, 확대된 지도에서 원이 화면을 과도하게 덮는 문제를 수정했다.
- 운영 코드는 지역 집계 UI가 유지되고 있었지만 전환 기준이 너무 이른 레벨 5였음을 확인했다. 중간 줌에서도 `개수 + 지역명`을 유지하도록 레벨 3으로 조정하고 CSS/JS에 버전을 붙여 이전 캐시가 롤백처럼 보이는 문제를 방지했다.
