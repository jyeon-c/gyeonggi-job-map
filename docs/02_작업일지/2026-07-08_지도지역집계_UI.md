# 지도 지역 집계 UI 개선 (2026-07-08)

- 지도에 개별 채용공고 핀 대신 현재 지도 범위·검색·필터 결과를 첫 근무지역 기준으로 묶어 `공고 수 + 지역명` 배지로 표시했다.
- 여러 공고가 있는 지역 배지는 클릭 시 해당 위치로 확대하고, 단일 공고 배지는 기존처럼 공고를 선택하도록 했다. 카드 선택 시 소속 지역 배지도 선택 상태로 강조된다.
- PC에서 왼쪽 공고 카드에 마우스를 올리면 지도에 해당 좌표를 반투명 원으로 표시한다. 정밀 좌표는 700m, 지역 근사 좌표는 2.2km 반경으로 정확도 차이를 반영했다.
