- 프로젝트: 일자리 맵(JOBABA MAP) 웹 서비스
- 목적: 특정 웹페이지별 '일자리 맵' 기능 **분양(이식)**

- 대상 환경: PC / 모바일 웹 모두
- 일정: 8월 테스트, 9월 초·중 오픈

## 데이터·좌표 처리 결정

- **방침: 지역 중심점 대체**(제외 아님). 파이프라인이 시·군 중심점 + 결정적 지터로 근사 좌표를 부여(`geocodePrecision=region_approx`).
- 최종 운영 대상 460건은 좌표 결측 0건이다.
- `기업좌표_샘플.csv`는 `BUSINO ↔ BIZ_NO`가 일치하는 공고에만 `MAP_COOR_Y(위도)/X(경도)`를 적용한다.
- 프런트는 `region_approx` 공고에 "위치 근사" 표기.

## 최종 요약 (2026-07-08)

- **요청 10건 중 9건 운영 완료, 1건(#3) 코드 완료·운영 설정 대기**. PC IP 위치를 완성하려면 Cloudflare에서 `Add visitor location headers`를 활성화해야 한다.
- 보너스: **관리자 통계**(`/admin.html`, Spring Security HTTP Basic 인증) — 원 요청엔 없으나 구현.
- 라이브: **https://jobmapkorea.com** (EC2 + Docker Compose + Cloudflare HTTPS).

## 후속 과제 (선택)

- 카카오 REST 키로 좌표 정밀화(현재 시·군 근사 → exact), 희망임금 파싱 정교화
- SSH 소스 내 IP 제한(AWS 보안그룹), Cloudflare SSL Full+origin 인증서
