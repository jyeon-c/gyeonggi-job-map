## 제약사항 (명세서 기준)

- 지도 라이브러리: 카카오맵(JS 키 본인 발급)
- 좌표: 주소 기반 지오코딩으로 파생 (원천 DB 실시간 연동 아님, 제공 CSV 사용)
- 실 DB 미연동 = 고용24/잡코리아 원천 시스템 DB에 라이브 연결 금지. 자체 DB 설계·구축은 가능.
- 산출물에 실제 개인정보 포함 금지 → 제공 CSV는 이미 마스킹된 상태, 원본 파일 자체를 노출하지 않도록 주의

## 공공/민간 구분 (FR-02 관련)

- 공공: 고용24 (좌표 없음, `BASIC_ADDR` 100% 확보)
- 민간: 잡코리아 (좌표 없음, `AREA_INFO`만 존재, 사업자번호 매칭 시 정확주소 확보율 약 24%)
- 마커 색상/뱃지로 구분 표시
