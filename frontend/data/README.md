# 경기도 지도 집계 경계 데이터

- 파일: `gyeonggi-map-regions.geojson`
- 원천: 통계청 행정구역 경계
- 변환·공개: [SeoulTech/korea-maps](https://github.com/SeoulTech/korea-maps)
- 라이선스: Eclipse Public License
- 적용: 전국 읍면동 경계 중 코드 `41`(경기도)만 추출해 지도용 42개 권역으로 그룹화
  - 수원·성남·안양·안산·고양·용인은 안정적으로 유지된 16개 일반구 단위
  - 나머지는 시·군 단위, `여주군`은 현재 명칭인 `여주시`로 보정

지도 배지 명칭과 hover 경계는 공고의 원본 지역 문자열이 아니라 공고 좌표가 포함되는 이 폴리곤을 기준으로 결정한다. 경계 단순화로 폴리곤 밖에 살짝 놓인 좌표는 가장 가까운 권역으로 보정한다.
